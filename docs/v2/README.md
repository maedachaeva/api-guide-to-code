# v2. SW 활용 현황 API 가이드 설계

### 개요
로그인 성공 이벤트를 기준으로 SW 활용 현황을 분석하기 위한 통계 REST API를 설계했습니다.  
이벤트 기반 집계 구조를 통해 다양한 사용자 행동 지표로 확장 가능하도록 정의했습니다.

<br>

### API 설계 문서 (GitBook)
🔗[SW 활용 현황 API 가이드](https://sw-usage-api.gitbook.io/sw-usage-api-docs-docs/)에서 확인할 수 있습니다.

<br>

### 제공 API 목록
- 월별 접속자 수 조회 API
- 일자별 접속자 수 조회 API
- 평균 하루 로그인 수 조회 API
- 휴일을 제외한 로그인 수 조회 API
- 부서별 월별 로그인 수 조회 API

<br>

### 설계 기준
- 재사용성을 고려한 공통 파라미터 및 응답 구조 적용
- 접속자 수는 유니크 사용자 기준으로 집계
- 로그인 수 관련 지표는 로그인 성공 이벤트 건수 기준으로 집계
- 기간 조회 범위를 유연하게 설정할 수 있도록 from / to 파라미터 기반 API 설계

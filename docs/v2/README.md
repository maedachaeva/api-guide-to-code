# v2. SW 활용 현황 API 가이드 설계
| API 가이드 문서는 [SW 활용 현황 API 가이드](https://sw-usage-api.gitbook.io/sw-usage-api-docs-docs/)(GitBook)에서 확인할 수 있습니다.


### 제공 API 목록
- 월별 접속자 수 조회 API (MAU)
- 일자별 접속자 수 조회 API (DAU)
- 평균 하루 로그인 수 조회 API
- 휴일을 제외한 로그인 수 조회 API
- 부서별 월별 로그인 수 조회 API

<br>

### 개요
로그인 성공 이벤트를 기준으로 SW 활용 현황을 분석하기 위한 통계 API를 설계했습니다.<br>실제 서비스 운영에서 활용 가능한 지표를 REST API 형태로 정의했습니다.

<br>

### 설계 기준
- 재사용성을 고려한 공통 파라미터 및 응답 구조 설계
- 로그인 성공 이벤트를 기준으로 지표 정의
- 접속자 수(MAU/DAU)와 로그인 수(이벤트 건수) 기준 분리
- 기간 조회 범위를 유연하게 설정할 수 있도록 from / to 파라미터 기반 API 설계
- 통계 화면 활용을 고려한 응답 구조 설계
